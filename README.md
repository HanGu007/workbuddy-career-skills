## 核心能力

### 1. 简历专属诊断 — 拿到就能照着改的操作手册

多策略分层识别简历内容，输出A4单页HTML诊断报告（必出产物）。

- **多策略识别**：模型多模态直读 PDF/Word/图片 → 图片元素专项提取（印章/证书/证件照/数据图表）→ 信息遗漏自检与追问，确保文字+视觉信息零遗漏
- **四维度专属诊断**：人物画像（行业+岗位定位）→ 经验段位（应届/新人/中级/资深/管理层）→ 三大具体问题（定位到第X条经历+原文引用+HR视角解读+具体改写方向+🔴🟠🟡严重度标签）→ 被埋没的亮点（附"如果改写成XX，HR会看到YY"型改写建议）
- **诊断报告干货铁律**：①禁用通用废话（"缺乏量化""描述太弱"类评语一律禁止，必须引用原文片段）②每条问题必带改写方向 ③HR视角代入（解释"HR看到这句话时会怎么想"）④行动指令化（用祈使句而非建议句）
- **HTML报告**：7大模块（报告头部/人物画像/四维评分/三大问题/被埋没亮点/优化优先级/页脚），评分卡片含颜色进度条，单页紧凑排版

### 2. 简历逐条精准改写 — 6条红线 + 专属性自检

工作经历逐条四段式改写（原文→问题→优化→改动说明），附带个人简介四句式写作。

- **6大禁用词绝对替换**：负责/参与/协助/配合/完成/处理 → 主导/深度介入/统筹/交付/推进，附完整替换表
- **专属性自检机制**：每条改写后必自问"这句话能不能套用在另一个人身上？"，确保每句话只属于你
- **数据真实性底线**：简历中原有数字保留强化，绝不自编数据（改用规模/结果描述替代），主动追问用户回忆估算
- **段位匹配**：应届用执行者语气、5年+体现专业深度、管理层突出战略/团队，禁止过度包装
- **岗位专属匹配**：提供JD时自动执行关键词嵌入+职责对应+成果对齐+个人简介定向重写，实现"一岗一版"
- **四句式个人简介**：身份定位 → 核心能力 → 价值主张 → 差异化亮点，禁用"本人/希望/具有良好的/能够胜任"等模板词

### 3. 从零生成简历 — 8大模块引导式提问

适合尚无简历或想重建简历的用户，分2-3轮自然对话逐步收集信息。

- **结构化引导**：基本信息→专业技能(硬/软)→教育经历→工作/实习经历→项目经历→荣誉奖项→证书→意向岗位
- **经历提炼引擎**：对每段口述经历挖掘2-3个可量化成果点（处理了多少？提升了多少？节省了多少？），自然转化为STAR叙事
- **岗位专属生成**：提供JD时自动执行关键词预埋+经历排序调整+个人简介定向+50字打招呼用语（适配Boss直聘等平台）

### 4. 企业深度背调 — 8步流程 / 10章报告 / 3大核心风险

求职是双向选择——入职前看清公司的薪资、仲裁、征信三大命门。

- **薪资发放风险评估**（🆕核心）：资金链健康度（实缴比例/财务数据/参保趋势）→ 发薪稳定性交叉验证（欠薪诉讼/社保合规性）→ 裁员/业务收缩信号
- **劳动仲裁专项审查**（🆕核心）：近1年/累计劳动仲裁数量+案由类型分布（欠薪/违法解除/社保纠纷）+集体性欠薪检测
- **法人/高管征信深度审查**（🆕核心）：失信被执行人/限消令/被执行记录+高管曾任职企业风险+名下其他公司经营状态
- **8步完整流程**：工商核实→法人征信→劳动仲裁→薪资风险→股权架构→行业竞争力→异常信号扫描→综合评级
- **风险即时预警**：发现失信/欠薪黑名单/集体诉讼/股权大面积冻结等🔴高风险信号时立即中断流程、优先告知
- **多公司对比模式**：无目标时可基于职业画像智能推荐3-5家匹配公司+逐家背调+投递优先级排序
- **输出**：10章节结构化背调报告（含薪资风险评级/求职者必知清单）+ HTML文件

### 5. 面试辅导与模拟 — 基于简历的定制化深度问题

基于目标公司+岗位+你的简历，设计面试官视角的深度问题并模拟陪练。

- **前置简历深度解析**：识别简历中每个可能被HR/面试官追问的点（核心项目/关键数据/时间线异常/岗位跨度/自我评价承诺）
- **4维度面试问题设计**：①经历真实性与细节深挖（验证简历可信度，递进式追问）②能力匹配度考察（情境题，匹配岗位要求）③岗位适配性判断（探究转型逻辑/求职动机）④职业稳定性评估（空窗期/频繁跳槽等时间线异常解释）
- **每道问题含三重标注**：💡考察意图（这个问题在验证什么）+ 📝建议应答方向（STAR框架要点）+ 🎯追问策略
- **模拟面试+即时反馈**：逐题模拟→用户回答→亮点✅/改进⚠️/参考话术📚三层反馈
- **输出面试手册**：公司调研摘要+背调摘要+深度问题清单+自我介绍优化版+反问问题建议

### 6. 知识库检索引擎 — 726条专业资料的强制调用基石

强制调用IMA知识库「AI简历助手」，确保每条诊断和改写都有HR实战案例支撑。

- **指定知识库**：AI简历助手（ID: 7467810531846595 / 726条内容），含简历模板/STAR法则/面试真题/B站职场博主视频转录等
- **前置验证协议**：每次会话首次简历场景自动执行连通性验证，验证失败触发三级严重警告+持续降级通知
- **批量多维检索**：每个环节3-5个关键词组合+同义词扩展+行业场景关键词，简历诊断≥10篇、面试准备≥20篇
- **文件来源透明**：每次检索必须列出匹配到的具体文件名，不得只说"参考了知识库"

### 7. 简历HTML生成 — A4单页精确排版 + PDF零偏差

将优化后的简历生成为A4单页HTML，附带PDF转换和一页适配保障。

- **精确物理尺寸**：180mm×267mm内容区，48行上限，字体栈 PingFang SC / Microsoft YaHei，单位全部使用mm/pt（禁止px/em）确保打印零偏差
- **10级字体层级**：姓名22pt→模块标题11pt→正文9.5pt→标签8.5pt，信息密度与可读性精确平衡
- **一页适配自检**：行数估算→内容溢出时按精简优先级处理（项目经历>荣誉证书>工作经历条目>教育经历>个人简介）
- **PDF转换保障**：@page size: A4 + print-color-adjust: exact + preferCSSPageSize: true，杜绝乱版
- **岗位专属版**：提供JD时生成定向HTML+50字打招呼用语+经历重排序
- **配色方案**：深蓝黑#1a1a2e + 紫色强调#534AB7 / 浅紫标签背景#F0EEFF

### 8. 求职档案追踪 — Excel + JSON双通道同步

管理投递全生命周期，从首次投递到Offer薪资对比。

- **4表结构Excel模板**：投递总览(仪表盘) / 投递明细表(17字段) / 面试准备清单 / 薪资对比分析
- **10类自动同步场景**：简历优化完成→背调完成→面试准备完成→状态变更→收到Offer等，自动写入对应Excel位置
- **HTML可视化看板**：关键指标卡(总数/活跃/面试中/Offer)+漏斗图+状态分布+活跃投递列表+待办提醒
- **7种状态流转**：待回复→笔试→一面→二面→HR面→Offer→已拒绝，附自动状态更新规则
- **隐私与安全**：写入前备份(.bak)、追加不覆盖、状态列可更新、双通道JSON+Excel互备

### 9. 知识卡片生成 — 干货提取 + 精美设计 + 自动截图

将任何文字材料转化为3:4比例（900×1200px）的精美知识卡片组，自动截图保存。

- **三步流水线**：内容分析提取干货→分章节规划卡片(封面/章节引导/内容/总结)→生成HTML+Playwright自动截图PNG
- **视觉规范**：5套配色方案(商务/成长/清新/科技/人文)，正文≥18pt/标题≥24pt/关键数据≥28pt大字号，3:4比例适配手机收藏
- **干货密度原则**：每卡1个核心观点+2-4个支撑要点，不留大片空白，装饰编号48-72px低透明度填充
