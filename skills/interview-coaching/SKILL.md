# 面试辅导与模拟

> 本 Skill 为「职途星」专家的核心能力模块，用于基于目标公司和岗位信息设计针对性面试问题、准备STAR行为面试答案、模拟面试场景。

## 触发时机

当用户进入面试准备环节时启用（路径A完成简历优化后自动进入，路径C专门面试辅导时触发）。

## 一、简历深度解析（前置步骤）

收到用户简历后，首先深度解析简历内容，识别以下关键要素（必须在后续面试问题设计中体现）：

- **核心项目/经历**：用户最重点描述的2-3个项目或工作经历
- **关键数据点**：简历中出现的所有数字、百分比、金额、规模等
- **技术栈/工具**：提到的专业工具、平台、方法论
- **时间线**：各段经历的连贯性，识别空窗期、频繁跳槽等潜在关注点
- **岗位跨度**：是否有跨行业/跨职能的经历，需要解释转型逻辑
- **自我评价中的承诺**：个人简介中提到的能力，必须在面试问题中验证

## 二、面试官视角深度问题设计

从4个维度设计深度面试问题。每个问题必须：
- 针对简历中的**具体项目/数据/经历**展开，不能是通用问题
- 具有"追问感"——让面试者感受到"面试官确实认真读了我的简历"
- 标注**考察意图**——这个问题在验证什么能力或经历真实性

### 维度一：经历真实性与细节深挖（验证简历可信度）

- 针对简历中最亮眼/最可疑的项目，设计3-5个递进式追问
- 问题模板："你提到主导了[XX项目]，当时[具体场景]是什么？你[具体做了什么]让结果优于预期？最终的[具体指标]是多少？"
- 考察意图：验证项目真实性、考察细节掌握度、识别"包装过度"

### 维度二：能力匹配度考察（匹配岗位要求）

- 针对JD中的核心能力要求（如有）或岗位通用能力模型，设计2-3个情境题
- 问题模板："如果现在面对[具体场景]，你会怎么[具体动作]？"
- 考察意图：验证实际工作能力、考察思维方式、评估经验迁移能力

### 维度三：岗位适配性判断（判断人岗匹配度）

- 针对用户背景与目标岗位的差异点，设计1-2个探究性问题
- 问题模板："你上一份工作是[XX行业/岗位]，这次应聘[YY岗位]，[转型逻辑]你是怎么考虑的？"
- 考察意图：评估求职动机、判断稳定性、了解职业规划清晰度

### 维度四：职业稳定性评估（判断长期潜力）

- 针对简历中的时间线异常（空窗期、频繁跳槽等），设计1-2个解释性问题
- 问题模板："你[XX工作]是[结束时间]，[YY工作][开始时间]，中间[空窗期时长]主要做了什么？"
- 考察意图：评估稳定性、了解真实离职原因、判断是否有隐藏问题

## 三、问题输出格式

```
## 🎯 面试官深度问题清单（基于你的简历定制）

### 一、经历真实性与细节深挖（验证简历可信度）
1. **针对[简历中的具体项目]**：[具体问题，包含追问点]
   💡 **考察意图**：[这个问题在验证什么]
   📝 **建议应答方向**：[STAR法则框架下的回答要点]

### 二、能力匹配度考察（匹配岗位要求）
...

### 三、求职动机与岗位适配性（判断人岗匹配度）
...

### 四、职业稳定性评估
...
```

## 四、模拟面试与应答辅导

- 逐题模拟，用户回答后即时反馈
- 反馈框架：✅亮点 → ⚠️可改进 → 📚参考话术
- 帮助用户用STAR法则组织答案，确保回答有结构、有数据、有结果

## 五、输出面试手册

通过HTML文件输出面试准备文档，包含：
- 公司调研摘要（如有）
- 企业背调摘要（💡增强模式）
- 深度问题清单（含考察意图和建议应答方向）
- 自我介绍优化版本
- 反问问题建议（展现对公司和岗位的了解）
