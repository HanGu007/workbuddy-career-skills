# 简历逐条精准改写 + 个人简介写作

> 本 Skill 为「职途星」专家的核心能力模块，包含工作经历逐条改写规范和个人简介四句式写作规范。

## 触发时机

当简历诊断完成后，进入工作经历改写和个人简介写作环节时使用。

---

## 一、工作经历逐条改写（四段式结构）

对工作经历的每一条进行改写时，必须遵循以下结构：

```
▸ 原文：[保留用户写的原始内容，一字不改]
▸ 问题：[这句话具体哪里弱，一句话说清楚]
▸ 优化：[改写后的版本]
▸ 改动说明：[你改了什么、为什么这样改]
```

### 改写四大核心规则

#### 规则1 — 禁用词（绝对执行，零容忍）

以下词语**一律不得出现**在优化版本中：
> **"负责"、"参与"、"协助"、"配合"、"完成"、"处理"、"从事"**

替换方案（具体动词替代模糊动词）：
| 禁用词 | 替换为 |
|--------|--------|
| 负责 | 主导、统筹、管控、执行、推进 |
| 参与 | 深度介入、全程跟进、担任XX角色 |
| 协助/配合 | 支撑、协同推进、配合输出（用于次要角色）→ 能改主动就改主动 |
| 完成 | 交付、落地、推动、实现 |
| 处理 | 处置、跟进、解决、应对 |
| 从事 | 专注于、深耕、聚焦 |

#### 规则2 — 数据原则（真实性底线）

- 用户简历中有数字：**保留并强化**，放到句子的显眼位置
- 用户简历中没有数字：**不得自行编造**，改用规模或结果描述替代
  - ❌ 错误："销售额提升约30%"（用户从未提到这个数字）
  - ✅ 正确："持续优化转化路径，推动销售额阶段性增长"
  - ✅ 正确："主导客户开发工作，累计维护服务客户规模显著扩大"
- 如果用户说不出具体数字，主动追问引导其回忆估算，但绝不替用户编数字

#### 规则3 — 专属性检查（每条改完必须自问）

改完每条经历后，必须自问：**"这句话能不能原封不动地套用在另一个人的简历上？"**

- 如果可以 → 说明改得不够具体，必须重写，加入只属于这段经历的细节
- 如果不可以 → 通过检查

专属性要素参考：特定规模（团队人数/客户数量/金额范围）、特定工具/平台、特定行业背景、特定场景或挑战、特定成果或结果

#### 规则4 — 段位匹配（禁止过度包装）

- 3年经验的人：改出来就是有一定积累的执行者语气，不能写成听起来像"总监统筹全局"
- 5年经验的人：体现专业深度和项目管理能力
- 管理层：突出团队建设、战略落地、业务结果
- HR看到过度包装的简历，第一反应是不真实 → 拿到面试也会被当场戳穿

#### 规则5 — 岗位专属匹配（💡用户提供JD时执行）

当用户提供目标岗位JD时，改写必须针对该岗位进行精准匹配：
1. **关键词嵌入**：将JD中出现的核心技能、工具、方法论关键词自然嵌入改写内容，但不生硬堆砌
2. **职责对应**：用户经历中与JD核心职责相关的条目，优先排序、优先强化描述
3. **成果对齐**：优先突出与JD关注指标一致的成果（JD强调增长则突出增长数据，JD强调降本则突出成本优化）
4. **个人简介重写**：四句式结构中，第二句（核心能力）和第三句（价值主张）必须直接回应JD的核心需求
5. **区分不同岗位**：同一用户的底稿经历，投递不同岗位时，改写侧重完全不同——这是"一岗一版"的核心价值

### 简历书写红线

**✅ 必须做到**：
1. **行动动词开头**：每一条工作描述都以强有力的行动动词开头
2. **量化嵌入**：尽可能使用具体数字和百分比来量化成就
3. **STAR叙事流畅**：运用STAR法则组织内容（情境→任务→行动→结果），但以自然叙事呈现，不露痕迹
4. **简洁有力**：每条描述控制在1-2行，信息密度高，拒绝冗长废话

**❌ 绝对禁止**：
1. **禁止出现"我"字**：直接以动词开头
2. **禁止STAR标签词**：不得出现"情境""任务""行动""结果"等标签词
3. **禁止"任务是…"句式**
4. **禁止纯职责罗列**：每条描述必须包含成果
5. **禁止虚构数据**：所有量化数字必须来自用户确认

### 优秀工作描述示例

- 参与某大型地方国企客户融资pitchbook撰写，梳理归纳包括可交换债券、可转换债券、优先股、资产支持证券等在内的相关金融产品的定义、特征、优缺点及应用的经典实例，交付21页幻灯片并被团队采纳使用
- 负责美国原油基金（United States Oil Fund LP）的专题研究，通过阅读数百页全英基金招募说明书、年报、市场简报等文件，整理分析其运作主体、运作机制、市场表现、同业竞对等，独立撰写8页基金分析报告
- 根据JD主动搜集并筛选简历，通过电话与求职者沟通，以候选人求职的痛点为切入点，邀约参加面试。日均处理简历量60+，邀约率平均65%，到面率平均70%+
- 负责中西部城市实习生招聘全流程工作，包括与业务部门沟通招聘需求、发布招聘信息、维护并拓展招聘渠道（BOSS直聘、应届生求职网等）、筛选简历、电话面试、办理入职等流程，累计处理近2000+简历，推荐152位候选人参加面试，产出45个offer

## 二、个人简介四句式写作

根据用户的真实经历，写一段3-4行的个人简介，遵循以下四句式结构：

1. **第一句 — 身份定位**：点明是谁（行业+年限+核心能力方向），简洁有力
2. **第二句 — 核心能力**：说擅长什么（必须具体，不是"具有良好的沟通能力"）
3. **第三句 — 价值主张**：说能给公司带来什么价值（结果导向，不是"希望能贡献"）
4. **第四句（可选）— 差异化亮点**：一句有个性的补充，让简介读起来像真人说话

### 个人简介禁用词

以下词语一律不准出现：
> **"本人"、"本着"、"致力于"、"希望"、"热爱"、"具有良好的"、"能够胜任"、"积极主动"**

**检验标准**：读起来像一个真实的人在说话，不像从模板复制粘贴出来的。
