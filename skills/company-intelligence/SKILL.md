# 目标公司情报分析

> 本 Skill 为「职途星」专家的核心能力模块，用于搜索目标公司的业务方向、产品线、组织架构、近期动态等关键信息，据此精准调整简历措辞和重点。

## 触发时机

💡用户提供目标公司名称后启用。用户未提供公司名称时不调用。

## 信息采集流程

### Step 1：锁定目标公司

调用企查查 `get_company_by_query` → 根据公司名称搜索，锁定目标公司

### Step 2：获取公司画像

调用企查查 `get_company_profile` → 获取：
- 公司简介
- 业务模式
- 行业分类
- 核心产品/业务线

### Step 3：获取组织与团队

调用企查查 `get_key_personnel` → 获取核心团队信息（有助于了解面试官背景和组织架构）

### Step 4：了解业务版图

调用企查查 `get_external_investments` → 获取对外投资信息（了解业务版图和战略方向）

### Step 5：天眼查补充（可选）

当企查查数据不够详细时，调用天眼查补充：
- `search_companies` → 按`名称`搜索公司，获取 company_id
- `get_company_basic_profile` → 获取公司基础画像
- `get_company_group_profile` → 获取集团画像，识别集团成员和投资方（适合大公司架构分析）

**优先级**：企查查优先（响应更快），天眼查作为补充和交叉验证

## 情报分析输出

将采集的信息整理为「公司情报摘要」，包含：

| 维度 | 信息 |
|------|------|
| 公司全称 | [从工商信息获取] |
| 所属行业 | [行业分类] |
| 主营业务 | [从公司简介提取] |
| 核心产品/业务线 | [列出主要产品或服务] |
| 组织架构线索 | [事业群/部门/核心团队] |
| 近期动态 | [融资/扩张/收缩/新品发布等] |
| 面试官背景线索 | [核心团队背景，有助面试准备] |

## 简历优化应用

情报分析完成后，据此调整简历优化方向：

1. **业务方向匹配**：如果公司主营B端SaaS，简历中B端产品经历要前置并强化
2. **技术栈对齐**：如果公司技术栈是Go+K8s，简历中相关技术经历要突出
3. **文化匹配**：如果公司是创业团队，突出创业经历和多面手能力
4. **面试官背景**：了解面试官的技术背景，准备对应深度的技术讨论
