# 简历从零生成

> 本 Skill 为「职途星」专家的核心能力模块，用于用户尚未制作简历时通过结构化引导生成专业级简历。

## 触发时机

当用户表示"我还没有简历"、"帮我从零开始做一份"时启用。

## 引导式提问（8大模块）

通过结构化问题逐步收集简历所需信息：

### ① 基本信息
姓名、性别、年龄、联系方式（手机号码和电子邮箱）、求职意向

### ② 专业技能
- 硬技能：与目标岗位JD相关的专业技能，如专业软件使用（Photoshop、AutoCAD等）、办公软件、编程语言、工具平台等
- 软技能：沟通能力、团队合作能力、书面表达能力、项目管理能力等

### ③ 教育经历
学校名称、专业、学历、在校起止时间、主要课程、在校期间的成绩和排名（如果优秀的话）

### ④ 工作/实习经历
- 公司名称、部门、职位、工作时间
- 主要工作内容
- 工作业绩和成果（引导量化：处理了多少？提升了多少？节省了多少？）
- 为每段经历挖掘2-3个可用数字衡量的成果点

### ⑤ 项目经历
- 项目名称和背景
- 你负责的内容和角色
- 项目成果和你的贡献（引导量化）

### ⑥ 荣誉奖项
时间、奖项名称、级别

### ⑦ 证书及其他
专业资格证书、语言证书、其他加分项

### ⑧ 意向岗位
目标岗位名称、目标公司名称（非强制，提供后可做企业背调）、目标岗位JD（非强制，提供后可做精准匹配优化）、期望行业/城市/薪资

## 经历提炼规则

将用户口述的经历转化为专业简历语言时，必须：

1. **挖掘可量化的成果点**：对每段经历追问2-3个数字
2. **遵循书写红线**（详见 resume-rewrite Skill）：
   - 行动动词开头
   - 量化嵌入
   - STAR叙事流畅但不露标签
   - 禁止"我"字、禁止"任务是"句式、禁止纯职责罗列
3. **运用STAR法则组织内容**，但以自然叙事呈现
4. **段位匹配**：应届/新人写法不同于资深/管理层

## 生成结构

整合所有信息后，按以下结构生成简历（具体排版由 resume-html-generator Skill 执行）：

```
## [姓名]
[性别]/[年龄]/[政治面貌（可选）]
[手机号码] | [电子邮箱]
[求职岗位]

### 教育经历
#### [学校名称] - [专业] | [毕业年份]

### 工作/实习经历
#### [职位名称] | [公司名称] | [起止日期]

### 项目经历（如有）
#### [项目名称] | [角色] | [起止日期]

### 专业技能
### 荣誉奖项（如有）
### 证书及其他（如有）
```

## 岗位专属简历生成（💡用户提供目标公司+岗位JD时）

当用户提供了目标公司和岗位JD时，简历生成必须针对该岗位做专属优化：

1. **求职意向精准化**：`[求职岗位]` 直接写目标岗位名称，不再写笼统方向
2. **经历排序调整**：与JD核心职责最相关的经历排在最前面
3. **关键词预埋**：将JD中的核心技能、工具、方法论关键词自然融入工作描述
4. **个人简介定向**：四句式个人简介中的"核心能力"和"价值主张"两句直接回应JD核心需求
5. **文件命名后缀**：输出的HTML/PDF文件名加 `_[公司名]_[岗位名]` 后缀
6. **打招呼用语**：同步生成50字以内打招呼用语（见下方规范）

## 打招呼用语生成规范

每份岗位专属简历必须附带一句打招呼用语：

- **字数**：50字以内（含标点），适配Boss直聘/智联招聘等平台的字数限制
- **内容结构**：我是谁（年限+领域）+ 核心匹配点（1个JD最看重的）+ 行动号召
- **禁忌**：
  - ❌ 禁止套话："您好，我对贵公司很感兴趣，希望能有机会面试"
  - ❌ 禁止空洞："我有丰富的经验，期待与您沟通"
  - ✅ 必须有针对该岗位的具体匹配点
- **语气**：专业但不生硬，有温度但不谄媚
- **输出格式**：
  ```
  💬 打招呼用语（[公司名]-[岗位名]）：
  [50字以内打招呼内容]
  ```
- **示例**：
  - 快消方向："您好，3年快消品牌营销经验，主导过2个千万级新品上市，擅长用户增长与渠道投放，期待与您进一步沟通！"
  - 技术方向："您好，5年Java后端开发经验，主导过日活百万级微服务架构改造，熟悉Spring Cloud全栈，期待交流！"
  - 运营方向："您好，4年社区运营经验，从0到1搭建3个10万+用户社区，擅长用户分层与活跃度提升，期待沟通！"

## 交互原则

- 不一次性抛出所有问题，分2-3轮自然对话收集
- 对每段经历都要追问可量化的成果点
- 如果用户暂时无法提供某些模块信息，先跳过继续，后续补充
- 温馨提示：💡 如果你后续确定了目标公司和岗位，随时告诉我——我可以帮你做精准匹配优化和企业背调
